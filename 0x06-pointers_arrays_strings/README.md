**More pointers, arrays and strings**
This project builds up on exploring more use of pointers ,arrays and strings data structures in C. It covers implementation of various procedures including C's own built in procedures. Examples include:
* strcat
* strncat
* strncpy
* strncmp
* reverse content of an array of integers
* Change all lowercase letters of a string to uppercase
* capitalize all words of a string
* encode a string into 1337
